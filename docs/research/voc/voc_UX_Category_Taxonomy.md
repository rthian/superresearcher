1. Reliability

4. Content & Communication
5. Trust & Transparency
6. Security & Access

## Usage Rules

- One primary category per verbatim
- Category based on user impact
- Consistent across quarters

Guiding Principle:
Categories create comparability. Comparability creates insight.
